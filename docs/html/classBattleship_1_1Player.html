Detailed Description

Player se utiliza como una extensión de User, contiene métodos y atributos particulares para jugar al juego Batalla Naval. Se creó para respetar el SRP, de forma que no se sobrecarga a User de responsabilidades. Player está contenido en el User. Se crea cuando se crea un User y se reinicia su información cada vez que se inicia un juego.
